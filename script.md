# 朝礼スピーチ台本（約3分）— azukiazusa

---

【スライド1：タイトル】
おはようございます。azukiazusa です。
今日は、最近登山にハマっているという話をしたいと思います。

【スライド2：3つの山】
ここ最近、週末に3つの山に登りました。高尾山、宝登山（ほどさん）、そして大山です。高尾山は観光地としても有名な山で、名前を聞いたことがある人も多いと思います。宝登山は、長瀞（ながとろ）にある山で、登山初心者でも登りやすい山として知られています。長瀞といえば岩畳も有名ですね。そして大山は、神奈川県にある標高1,252mの山で、多くの登山客で賑わっていました。

【スライド3：登りやすい2座】
高尾山と宝登山は、道がよく整備されていて、初心者の自分でも、森の空気を楽しみながら気持ちよく登れました。

【スライド4：大山の試練】
高尾山の次は大山、という情報がネットにあって、それを信じて登ったのですが、これが大変でした。
ゴツゴツした岩山で、急な登りが延々と続きます。翌日には、見事にふくらはぎが筋肉痛になりました。下りのダメージが特に大きかったですね。

【スライド5：登山中の本音】
登っている最中の頭の中は、こんな感じです。
「疲れた…」「足が痛い…」「もう帰りたい…」「ぜんぜん楽しくない…」（ここは感情を込めと読む）
なぜわざわざ休みの日に辛い思いをしなければいけないのか、人間の営みとは大層不思議なものであります。

【スライド6：山頂の達成感】
それでもなんとか登り切るその瞬間、山頂で、苦しさが一気に吹き飛ぶような達成感が待っているのです。
これは、日常では決して味わえない感覚でした。
不確実性の高い社会では今自分がやっていることが正しいのかどうか、なかなか判断がつかないのですが、登山では目の前の山を登るというシンプルな行為が、目に見える形で結果として返ってくるというのも、登山の魅力のひとつでしょう。

【スライド7：学び】
最近は、AIのおかげで何でも効率よくこなせるようになりました。
でも、効率ばかりを追い求めていると、どこかで疲れてしまう気がします。
だからこそ、たまには自然とふれ合って、あえて不便さを味わうという、その「遠回り」の経験が、心の健康にとって大切なのではないかと思います。

みなさんもよかったら週末、近所の山に登ってみてください。
以上です。ありがとうございました。
